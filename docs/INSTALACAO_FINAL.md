# Instalação da versão Final 3.3

1. Entre em `frontend` e copie `.env.example` para `.env.local`.
2. Preencha a URL e a chave pública do Supabase.
3. Execute `npm install` e `npm run dev`.
4. Para produção, execute `npm run build`.
5. O SQL `database/auditoria_e_backup.sql` é opcional e deve ser revisado antes de executar.
